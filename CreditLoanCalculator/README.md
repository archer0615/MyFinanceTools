# CreditLoanCalculator

信用貸款試算工具，可直接在瀏覽器開啟使用，不需要安裝套件或啟動伺服器。

## 開啟方式

直接開啟 `index.html`。

## 主要功能

- 信用貸款月付金試算
- 總利息與總還款金額試算
- APR 概算
- 提前清償模擬
- 多筆提前清償比較
- 分段利率設定
- 浮動利率 / 指標利率模型
- DSR / 收入比與可貸額度反推
- 壓力測試與風險警示
- 銀行方案比較
- 轉貸 / 貸款重整比較
- 攤還明細、年度統計與圖表
- 分享頁快照

## 檔案說明

- `index.html`：主頁面
- `styles.css`：樣式
- `script.js`：主要試算邏輯
- `share.html`：分享頁
- `share.js`：分享頁邏輯

## 使用流程

1. 開啟 `index.html`
2. 填入貸款金額、年利率、期數與費用
3. 按下試算
4. 查看月付金、總利息、APR 與還款明細
5. 如有需要，切換到提前清償、風險分析或方案比較
6. 使用分享頁整理試算結果

## 單位

- 貸款金額：萬
- 提前清償金額：萬
- 月付金 / 本金 / 剩餘本金：萬
- 開辦費 / 手續費：元
- 提前清償手續費：元
- 利息：元
- 月收入：元

## 注意事項

本工具提供試算與比較用途，實際貸款條件、APR、費用與提前清償規則仍以銀行合約及正式核准結果為準。
