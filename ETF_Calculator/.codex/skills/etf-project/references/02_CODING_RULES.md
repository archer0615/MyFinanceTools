# Engineering Rules

## 禁止

- React
- Vue
- TypeScript
- npm
- backend
- API

---

## 必須

- Vanilla JavaScript
- file:// compatible
- offline-first

---

## Coding Style

- prefer pure functions
- no giant classes
- max function length: 80 lines
- max file length: 1500 lines

---

## Naming Convention

- camelCase for variables
- PascalCase for classes
- UPPER_CASE for constants
