# Source Migration Scaffold

This folder mirrors the target Prompt_v5 architecture while the runtime app remains in `frontend/`.

- `components/`: future reusable UI components
- `features/`: feature-level composition
- `engines/`: financial calculation engines
- `simulations/`: simulation orchestration
- `workers/`: worker entrypoints
- `charts/`: chart adapters and visualization models
- `models/`: domain models
- `services/`: data and integration services
- `utils/`: shared utilities
- `hooks/`: future framework-specific hooks
- `store/`: state management
- `types/`: TypeScript type exports
