# 使用者情境

此工具的使用者為：

- 一般購屋族
- 首購族
- 房貸轉貸族
- 投資型房貸使用者

工具目的：

- 比較不同房貸方案
- 分析現金流
- 評估提前還款效益
- 計算長期總利息
- 評估寬限期影響
- 分析升息風險

---

# UI 行為需求

## 首頁區塊

首頁需包含：

1. 貸款列表區
2. 新增貸款按鈕
3. 試算結果區
4. 總結區
5. 攤還表區
6. 比較分析區

---

# 貸款卡片 UI

每一筆貸款需顯示為獨立卡片。

卡片包含：

- 貸款名稱
- 貸款金額
- 年限
- 寬限期
- 還款方式
- 利率階段
- 提前還款設定

卡片需支援：

- 展開/收合
- 刪除
- 複製貸款
- 即時重新計算

---

# 即時計算

當使用者修改：

- 金額
- 利率
- 年限
- 寬限期
- 提前還款

系統需：

- 自動重新計算
- 不需按下提交按鈕
- 即時更新結果

---

# 組合式房貸規則

系統需允許：

- 多筆貸款同時存在
- 不同利率
- 不同年限
- 不同寬限期
- 不同提前還款設定

總結區需顯示：

- 總貸款金額
- 每月總月付
- 總利息
- 總還款金額
- 最後清償日期

---

# 多段利率邏輯

每個貸款需支援：

- 任意數量利率階段
- 使用者可新增/刪除利率階段

每段包含：

- 月數
- 年利率

規則：

- 月數加總不可小於貸款總月數
- 若不足，自動延伸最後利率
- 每段切換時重新計算剩餘本金

---

# 寬限期邏輯

寬限期間：

- 僅支付利息
- 不償還本金

寬限期結束後：

- 重新計算剩餘期數月付

---

# 提前還款邏輯

提前還款支援：

- 多次提前還款
- 任意月份

每筆包含：

- 月份
- 金額

提前還款模式：

1. 減少月付
2. 縮短年限

重新計算規則：

- 提前還款後立即重新攤還
- 不可出現負本金
- 若提前清償則提前結束

---

# 本息平均攤還規則

需正確計算：

- 每月利息
- 每月本金
- 剩餘本金

需避免：

- 浮點數誤差累積
- 最後一期殘值

金額：

- 四捨五入至整數
- 或保留小數兩位

---

# 本金平均攤還規則

每月本金固定：

```text
本金 = 總本金 / 總期數
```

利息隨剩餘本金遞減。

---

# 攤還表需求

攤還表需：

- 支援數千筆資料
- 可捲動
- 表頭固定

欄位：

| 月份 | 日期 | 月付 | 本金 | 利息 | 剩餘本金 |

需支援：

- CSV 匯出
- 一鍵複製

---

# 圖表需求

不可使用 chart library。

需使用：

- HTML Canvas
或
- SVG

需繪製：

1. 月付走勢圖
2. 本金/利息比例圖
3. 剩餘本金曲線
4. 提前還款影響圖

---

# 比較模式

使用者可建立多個方案。

例如：

- 方案 A：30 年
- 方案 B：40 年
- 方案 C：寬限期 5 年

比較區需顯示：

- 月付差異
- 總利息差異
- 清償時間差異

---

# 壓力測試

系統需模擬：

- 利率 +0.5%
- 利率 +1%
- 利率 +2%

並顯示：

- 新月付
- 增加幅度
- 增加百分比

---

# DSR 計算

輸入：

- 月收入
- 信貸
- 車貸
- 信用卡分期

輸出：

- DSR %
- 建議負擔等級

等級：

- 安全
- 注意
- 風險偏高

---

# localStorage 行為

系統需：

- 自動保存
- 自動恢復

保存內容：

- 所有貸款
- 所有方案
- UI 狀態

---

# 錯誤處理

需檢查：

- 貸款金額不可小於 0
- 利率不可負數
- 年限不可為 0
- 提前還款不可超過剩餘本金

錯誤需：

- 顯示於 UI
- 不可使用 alert()

---

# 效能要求

需可處理：

- 10+ 組貸款
- 480+ 月攤還
- 多方案比較

不可：

- 明顯卡頓
- 阻塞 UI
