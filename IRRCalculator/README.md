# IRRCalculator

純前端 IRR 與投資報酬分析工具，可直接用瀏覽器開啟，不需要後端服務或安裝套件。

## 功能

- 現金流 XIRR：依日期與現金流計算年化 IRR。
- 組合 IRR：可用備註分組，合併多標的現金流後計算投資組合 IRR。
- 報酬換算：將近 N 年總報酬率換算成年化報酬率。
- 批次換算：一次處理近 1、3、5、10 年總報酬。
- 多標的比較：比較多個標的的年化報酬、加權分數與排序。
- 風險指標：依月報酬序列估算年化波動率、最大回撤與夏普值。
- 圖表視覺化：顯示單筆、批次與多標的年化報酬比較。
- 匯入匯出：支援現金流與標的資料 CSV / JSON 匯入，並可匯出 CSV。
- 報表輸出：可匯出 HTML 報表，或開啟列印視窗輸出 PDF。
- 本機保存：使用 `localStorage` 保留輸入內容與結果狀態。

## 使用方式

1. 直接開啟 `index.html`。
2. 在「現金流明細」輸入日期、金額與備註。
3. 投入金額請輸入負數，例如 `-100000`。
4. 回收、配息、贖回或出售金額請輸入正數，例如 `120000`。
5. 按下「計算 IRR」查看結果。

## 現金流資料格式

CSV 範例：

```csv
date,amount,note
2021-01-15,-100000,首次買進
2021-07-20,3500,現金股利
2022-08-10,-50000,加碼
2024-12-30,168000,出售
```

JSON 範例：

```json
[
  { "date": "2021-01-15", "amount": -100000, "note": "首次買進" },
  { "date": "2021-07-20", "amount": 3500, "note": "現金股利" },
  { "date": "2022-08-10", "amount": -50000, "note": "加碼" },
  { "date": "2024-12-30", "amount": 168000, "note": "出售" }
]
```

## 多標的資料格式

CSV 範例：

```csv
name,return1y,return3y,return5y,return10y,monthlyReturns
0050,18,35,72,185,"2.1,-1.2,3.4,1.1"
SPY,22,48,91,210,"1.8,-0.6,2.9,1.4"
```

欄位說明：

- `name`：標的名稱或 ticker。
- `return1y`：近 1 年總報酬率，單位為 `%`。
- `return3y`：近 3 年總報酬率，單位為 `%`。
- `return5y`：近 5 年總報酬率，單位為 `%`。
- `return10y`：近 10 年總報酬率，單位為 `%`。
- `monthlyReturns`：月報酬率序列，使用逗號分隔，單位為 `%`。

## 計算邏輯

- XIRR 先使用 Newton-Raphson 迭代。
- 若迭代不穩定，會 fallback 到二分法估算。
- IRR 需要至少一筆負現金流與一筆正現金流。
- CAGR 使用總回收除以總投入後依期間年化。
- TWR 以投入資本基礎估算期間報酬累積。
- MWRR 使用 XIRR 結果轉為百分比。
- 年化報酬使用公式：`(1 + 總報酬率) ^ (1 / 年數) - 1`。

## 注意事項

- 本工具為估算與比較用途，不構成投資建議。
- 日期區間過短、金額方向不足或現金流極端時，IRR 可能無法收斂。
- 「抓取」ticker 報酬會透過 Yahoo Finance 查詢，需瀏覽器可連線。
- PDF 輸出使用瀏覽器列印功能，實際版面依瀏覽器設定而定。
