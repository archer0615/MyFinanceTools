# 使用指南

## 新專案 Recovery 啟動

1. 閱讀 `.codex/BOOTSTRAP.md`
2. 執行 `MASTER_PLAN.md`
3. 每個 phase 後執行 ArchitectureReview

## 已 Drift 專案

1. 閱讀 `RECOVERY_BOOTSTRAP.md`
2. 先做 drift analysis
3. 禁止直接新增 feature
4. 逐 phase recovery

## Autonomous Workflow

系統採 phase-based orchestration：

- modeling
- workspace
- scenario analysis
- explainability
- visualization

每個階段皆需 review gate。
